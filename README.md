# React Landing Page

Modern responsive landing page built with **React, Vite and Tailwind CSS**.
The project focuses on reusable components, responsive layout and modern UI patterns commonly used in SaaS websites.

---

## Live Demo

https://react-landing.vercel.app

---

## Features

* Responsive layout (mobile, tablet, desktop)
* Modern SaaS-style landing page design
* Animated mobile navigation (hamburger menu)
* Scroll-based animations
* Reusable React components
* Clean UI built with Tailwind CSS
* Sticky navigation bar
* CTA section and testimonials

---

## Tech Stack

Project built using:

* React
* Vite
* Tailwind CSS
* JavaScript (ES6+)

## Installation

If you want to run the project locally:

```bash
git clone https://github.com/YOUR_USERNAME/react-landing.git
cd react-landing
npm install
npm run dev
```

---

## Build

To create a production build:

```bash
npm run build
```
